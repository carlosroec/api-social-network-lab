# Social Network Lab

```
Node.js
ES6
Babel
Express
Express JWT
MongoDB (Atlas)
```

## Add a file .env at the project root with the following data
```
# Port
PORT=3000

# Debug
LOG_LEVEL='silly'

# MongoDB
MONGODB_URI='mongodb://localhost:27017/social-network-lab'

# JSON Web TOken
JWT_SECRET='shhhhh-this-is-a-secret'
```

## Project setup
```
npm install
```

### Hot-reloads for development
```
npm run dev
```

### Run for production
```
npm run server
```
